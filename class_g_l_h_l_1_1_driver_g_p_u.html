Detailed Description

DriverGPU class is and abstraction GPU methods using OpenGL. Basically, ensures that the drivers are properly loaded once per thread, and gather all method in one class to more class-oriented likely interface. Every method is called as the original method but without the initial gl, so for more information visit https://www.opengl.org docs.
